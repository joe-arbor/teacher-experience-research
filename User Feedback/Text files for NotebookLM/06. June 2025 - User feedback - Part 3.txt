Comment: Hi,

Step 1: Create a User Defined Field (UDF)
Navigate to:
System > User Defined Fields

Select Add User Defined Field.

Complete the configuration:

Related to: Select Students

Field Type: Select Drop-down

Field Name/Identifier: Enter Focus Five

Options:

Yes

(Optional: Add No or leave blank as default)

Under Permissions, ensure that appropriate staff (e.g. designated SLT or Data Team) can access and edit the UDF.

Click Save to finalise the field.

This step establishes the backend field required to drive the visual flag.

Step 2: Create a Linked Visual Flag
Still in the System area, locate either:

School Settings > Student Flags

or System > Configuration > Flags (depending on your Arbor version)

Click New Flag.

Complete the flag setup:

Name: Focus Five Flag

Description: e.g. Used to identify students in termly Focus Five cohorts

Type: Select User Defined Field

Linked UDF Field: Choose the Focus Five drop-down created in Step 1

Display Icon/Colour: Choose a distinct colour and icon that does not conflict with existing flags (e.g. green circle)

Tooltip (optional): Focus Five student – termly support initiative

Active: Ensure this option is ticked

Click Create to finalise the visual flag.

This step ensures the system will display a coloured flag automatically when a student’s UDF is set to “Yes.”

Step 3: Apply the Flag to Students
Option A: Individual Student Assignment
Navigate to the relevant student profile.

Scroll to the Custom Fields or User Defined Fields section.

Set the Focus Five field to Yes.

Save the change.

Confirm that the Focus Five flag now appears in the student profile header.

Option B: Bulk Assignment
Navigate to:
System > User Defined Fields

Select the Focus Five field from the list.

Use the Bulk Actions > Edit Values tool.

Select the students to be included in the Focus Five cohort.

Set their field value to Yes and apply changes.

This method is recommended for termly updates.

Step 4: Confirm Flag Visibility
To verify the flag is functioning correctly:

Open a register for a class containing a Focus Five student. The coloured flag should appear next to their name.
